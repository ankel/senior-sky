# senior-sky

## Usage:
````
mvn clean install
cd raptor
mvn exec:java
```
then open `scorpion/index.html` in your browser of choice. Tested on Chrome 43.0.2357.134 (64-bit)
and Safari 8.0.8 (10600.8.9)
